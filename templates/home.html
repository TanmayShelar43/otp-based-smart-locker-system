Frequently Asked Questions

What is the Smart Locker System?

+

Our Smart Locker System is a secure, OTP-based storage solution that allows users to access their lockers using a One-Time Password (OTP) sent to their registered mobile or email. It ensures enhanced security for personal and business storage needs.

How does the OTP-based access work?

+

When a user requests access, a unique One-Time Password (OTP) is sent to their registered mobile or email. The user must enter this OTP on the lockerâ€™s keypad to unlock the system. The OTP is valid for a short period to prevent unauthorized access.

Where can this system be used?

+

Our Smart Locker System is ideal for various locations, including:

  • Offices & Workplaces
  • Schools & Universities
  • Residential Buildings
  • Delivery & Package Pickup Stations
  • Gyms & Sports Centers

What happens if I enter the wrong OTP multiple times?

+

For security reasons, if an incorrect OTP is entered multiple times, the system temporarily locks the user out. This prevents unauthorized access attempts. The user can request a new OTP after a short waiting period.

Can multiple users share the same locker?

+

Yes! The admin can assign multiple users to a single locker, each receiving a unique OTP when they need access.

Can I access my locker from a remote location?

+

Yes! The system can be integrated with an online platform where users can generate OTPs remotely and monitor their locker activity.
